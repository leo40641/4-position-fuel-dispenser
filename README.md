# 4-position-fuel-dispenser
Control and automation of a fuel dispenser with 4 filling positions.

- Hardware:
   - OrangePi Win.
   - Serial Touch LCD 7".
   - Barcode Reader.
   - XBee Pro S3B.
- Software:
   - Armbian (Debian Linux)
   - PyCharm Professional
   - XCTU
- DataBase:
   - SQLite 
- Programming Language:
  - Python
